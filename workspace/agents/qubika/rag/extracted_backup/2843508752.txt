Guía para crear un servicio desde cero con Ruby on Rails. Se toma de base el procedimiento realizado para bancard-pix-service
Creación del proyecto base con el comando rails new, utilizando las opciones acordes a la necesidad del proyecto.

Es posible que se necesite instalar MySQL en el entorno local para poder inicializar el proyecto. En caso contrario, falla al momento de hacer bundle install.
