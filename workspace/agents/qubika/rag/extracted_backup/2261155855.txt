This document aims to serve as a guide for setting up a dockerized Bancard project in a personal development environment.

Requirements
To have a GitHub user with the necessary permissions to download Bancard projects. Request it from the team lead or tech lead.

To have a personal SSH key associated with the GitHub user (Connecting to GitHub with SSH).

Note: The personal SSH key should not use a passphrase and should be generated using the ed25519 algorithm.
To have Docker Desktop installed and running.
Install Docker Desktop on Mac

Install Docker Desktop on Linux

Set up the moove-it VPN following theVPN Configuration (DEPRECATED)document.

Instructions
Project with docker-compose
All the necessary information for setting up a project with docker-compose should be found in the README of that project. If not, we appreciate the initiative to correct the README.
