Introducción
Los DevContainers son una solución que permite crear entornos de desarrollo consistentes y reproducibles usando contenedores Docker. Esta configuración garantiza que todos los miembros del equipo trabajen en el mismo entorno, eliminando el problema de "funciona en mi máquina".
Beneficios de usar DevContainers:
Consistencia: Mismo entorno para todos los desarrolladores

Aislamiento: No contamina el sistema host

Reproducibilidad: Fácil setup en nuevas máquinas

Versionado: El entorno está versionado junto con el código

En general, usar este Pull Request como base:Pull Requestde los cambios, es la versión que está aplicada en commerce-portal.
