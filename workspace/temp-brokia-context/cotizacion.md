# 📌 Proceso de Cotización de Vehículos

**Área:** Comercial  
**Objetivo:** Obtener y comparar ofertas de las compañías aseguradoras para presentar al cliente la mejor alternativa disponible.

---

## 1. Descripción General del Proceso

Cada vez que un cliente nuevo solicita asegurar su vehículo, se debe realizar una cotización en **las siete compañías de seguros con las que trabajamos**.

Actualmente, este proceso se realiza de forma manual, ingresando individualmente al cotizador web de cada compañía y cargando los datos requeridos.

El objetivo es obtener una propuesta comparativa que permita analizar costos y beneficios para asesorar correctamente al cliente.

---

## 2. Datos necesarios para cotizar

En la mayoría de las compañías, los datos obligatorios son:

- Marca del vehículo
    
- Modelo del vehículo
    
- Año del vehículo
    
- Zona de circulación
    
- Fecha de inicio de vigencia (por defecto: fecha del día de cotización, salvo que el cliente solicite otra)
    

### Casos particulares

Algunas compañías requieren datos adicionales:

- **Mapfre** → Solicita cédula de identidad del asegurado.
    
- **Porto Seguro** → Solicita cédula de identidad del asegurado.
    

Por lo tanto, para estandarizar el proceso y evitar reprocesos, se recomienda solicitar siempre la cédula desde el inicio, aunque no todas las compañías la requieran.

---

## 3. Procedimiento Actual (Manual)

1. Ingreso al cotizador web de cada compañía.
    
2. Carga individual de los datos requeridos.
    
3. Generación de cotización.
    
4. Registro manual de precios.
    
5. Elaboración manual de un cuadro comparativo.
    
6. Envío de propuesta al cliente (generalmente por WhatsApp o PDF armado manualmente).
    

Este proceso:

- Insume tiempo considerable.
    
- Es repetitivo.
    
- Está sujeto a errores humanos.
    
- No escala fácilmente si aumenta el volumen de cotizaciones.
    

---

## 4. Elaboración del Cuadro Comparativo

Actualmente el cuadro comparativo se realiza manualmente y contempla principalmente:

- Precio mensual o anual.
    
- Tipo de cobertura.
    
- Diferencias básicas entre planes.
    

La información se organiza para visualizar rápidamente qué compañía presenta:

- El menor costo.
    
- Mejor relación precio / cobertura.
    
- Mejores beneficios adicionales.
    

---

## 5. Objetivo de Automatización

El objetivo estratégico es automatizar completamente el proceso de cotización.

### 5.1 Automatización esperada

El sistema deberá:

1. Ingresar automáticamente a los cotizadores.
    
2. Cargar los datos del cliente y vehículo.
    
3. Obtener los precios de cada compañía.
    
4. Generar automáticamente un cuadro comparativo.
    
5. Emitir un PDF profesional listo para enviar al cliente.
    

---

## 6. Requisitos del Documento Final (PDF de Cotización)

El documento generado automáticamente deberá incluir:

### 6.1 Encabezado con datos de la cotización

- Nombre del cliente.
    
- Cédula de identidad.
    
- Marca, modelo y año del vehículo.
    
- Zona de circulación.
    
- Fecha de cotización.
    
- Fecha de inicio de vigencia.
    

### 6.2 Cuadro comparativo de precios

- Nombre de cada compañía.
    
- Precio mensual.
    
- Precio anual.
    
- Tipo de plan.
    
- Diferencia porcentual entre opciones (opcional en versión avanzada).
    

### 6.3 Resumen de beneficios diferenciales

Además del precio, el documento debe mostrar brevemente:

- Principales beneficios de cada compañía.
    
- Diferencias en:
    
    - Asistencia en viaje.
        
    - Vehículo de reemplazo.
        
    - Cobertura de cristales.
        
    - Deducibles.
        
    - Beneficios adicionales.
        

El objetivo es que el cliente no solo compare precio, sino también valor.

---

## 7. Resultado Esperado

Con la automatización del proceso:

- Se reduce el tiempo de cotización.
    
- Se disminuyen errores.
    
- Se mejora la presentación profesional.
    
- Se incrementa la capacidad de respuesta.
    
- Se facilita el crecimiento de la cartera.
    
- Se estandariza el asesoramiento.
