# Writer Style Guide — Brokia Tesis

## Objetivo
Redactar documentación académica formal, clara y natural, con voz de equipo técnico-profesional.

## Tono
- Formal, pero legible.
- Evitar jerga innecesaria y frases grandilocuentes.
- Priorizar trazabilidad: qué se decidió, por qué, con qué impacto.

## Estructura sugerida
1. Contexto
2. Decisión / Implementación
3. Evidencia / Resultado observable
4. Riesgos / Limitaciones
5. Próximos pasos

## Reglas de estilo
- Español rioplatense neutro.
- Párrafos cortos.
- Usar bullets para listas operativas.
- Incluir fechas y nombres de componentes (canal, job, herramienta) cuando aplique.
- Explicitar supuestos y datos faltantes con `[pendiente]`.

## Criterio académico
- Enfocar en método y proceso, no solo en resultado.
- Documentar iteraciones y cambios de diseño.
- Dejar base reutilizable para capítulo metodológico y anexos.
